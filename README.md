## Usage

```bash
npx esho hello.ts

```

#### Install globally

```bash
npm i -g esho

esho index.ts
```

#### Install as dependency

```bash
npm i esho
```

```json
{
  "scripts": {
    "start": "esho index.ts"
  },
  "dependencies": {
    "esho": "*"
  }
}
```
